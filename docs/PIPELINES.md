# Pipelines

## Recommended Depth Source

For most reconstruction workflows, FoundationStereo or Fast-FoundationStereo
is recommended over Quest native depth.

Stereo-based depth estimation typically provides:

- higher effective depth resolution
- denser depth maps
- fewer holes and missing regions
- improved alignment with RGB imagery
- higher-quality TSDF reconstruction results

Quest native depth remains useful when:

- stereo inference is unavailable
- maximum processing speed is required
- native device depth is specifically desired

## Pipeline Architecture

Capture Source
    -> Color Frames
    -> Depth Generation
    -> RGBD Integration
    -> TSDF Reconstruction

## Recommended Capture Source

MRUK capture is recommended for new datasets.

Benefits:

- official Meta capture format
- reliable pose-to-image correspondence
- consistent frame alignment
- better compatibility with stereo reconstruction workflows
- actively supported workflow

Legacy capture remains supported for compatibility with existing datasets.

Because Legacy datasets rely on timestamp-based synchronization between color,
depth, and pose streams, reconstruction quality may be affected by frame timing
mismatches.

## Capture Sources

### Legacy

Original Meta Quest capture format.

### MRUK

MRUK scene capture format.

## Depth Sources

### FoundationStereo / Fast-FoundationStereo (Recommended)

Stereo depth generated from left/right RGB images.

### Quest Native Depth

Depth maps provided by the Quest device.

## Reconstruction Modes

### Stereo ONNX Depth (Recommended)

Stereo depth generated by FoundationStereo or Fast-FoundationStereo is
converted into color-aligned RGBD frames and integrated using TSDF
reconstruction.

This workflow generally produces the highest-quality meshes and point clouds.

### Quest Native Depth

Quest depth maps are integrated directly into the reconstruction pipeline.

This workflow is faster and requires no stereo inference, but typically
produces lower reconstruction quality than stereo-based depth estimation.
