# Usage

Usage documentation has been split by audience so command-line workflows and Python API workflows can evolve independently.

- [CLI.md](CLI.md): command-line usage for normal pipeline users.
- [API.md](API.md): public Python API usage for downstream projects.
- [DATA_FORMAT.md](DATA_FORMAT.md): dataset layouts, timestamp conventions, coordinate systems, MRUK files, and generated artifact formats.

For installation details, see [INSTALLATION.md](INSTALLATION.md). For pipeline selection, see [PIPELINES.md](PIPELINES.md). For stereo model details, see [MODELS.md](MODELS.md).
